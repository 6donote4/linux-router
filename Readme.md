 This is a fork of [create_ap]().
 Share your Linux's Internet access to other devices.
 
Usage

```
# lnxrouter --ap wlan0 MyAccessPoint --password MyPassPhrase
```

Features

- Create Wifi hotspot and share Internet via NAT method
- DNS server
- DHCP server
